1. Click the big "New app" button
2. You'll see a form with these fields:
   - **Repository:** Select `Navaprabhas/document-chat-app`

3. Watch the deployment logs

3. Enter a test API key (or your own)

## 🐛 If Something Goes Wrong

### Build Fails
- Check Streamlit Cloud logs
- Verify `requirements.txt` is correct
- Check Python version compatibility

### App Loads But Errors
- Check if all dependencies installed
- Verify file paths are correct
- Check Streamlit Cloud logs for errors

### API Key Issues
- Make sure users enter valid Google Gemini API key
- Check Google AI Studio is accessible
- Verify API key has correct permissions
